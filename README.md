# Holistics Monaco Playground

A Vue 3 + Vite playground for customizing Monaco Editor styling with live CSS variable overrides.

## Stack

- Vue 3.5 + TypeScript
- Vite 6 + Tailwind CSS 4.1
- Monaco Editor (via monaco-editor-vue3)
- motion-v for animations

## Getting Started

```bash
npm install
npm run dev
```

## Scripts

| Command | Description |
|---------|-------------|
| `npm run dev` | Start dev server |
| `npm run build` | Type-check and build for production |
| `npm run preview` | Preview production build |
| `npm run type-check` | Run TypeScript type checking |

## Features

- **Home** — Animated landing page
- **Monaco Customizer** — Live editor with a sidebar to tweak 23 CSS variables and 5 editor options. Includes reset and CSS export.
